# 🚗 DriveSure AI — Vehicle Damage Segmentation Project

A custom YOLOv8 instance segmentation system trained on human-annotated car damage masks, featuring an interactive claims portal with a light/dark theme switcher and a guided claims chatbot.

---

## 📁 Project Structure

```
masks_human_project/
├── app.py                  ← Streamlit dashboard (run this!)
├── prepare_dataset.py      ← Dataset extraction & YOLO label converter
├── train_segmentation.py   ← YOLOv8-seg model training pipeline
├── predict_segmentation.py ← Inference engine (used by app.py)
├── chatbot.py              ← Guided claims assistant & FAQs
├── dataset.yaml            ← Generated by prepare_dataset.py
├── best.pt                 ← Trained YOLOv8s-seg model weights
├── requirements.txt        ← Python dependencies
└── data/
    ├── images/
    │   ├── train/          ← 80% training images
    │   └── val/            ← 20% validation images
    └── labels/
        ├── train/          ← YOLO polygon .txt annotations
        └── val/
```

---

## 🚀 Quick Start

### Step 1 — Install Dependencies
```bash
pip install -r requirements.txt
```

### Step 2 — Prepare Dataset
```bash
python prepare_dataset.py
```
This extracts binary masks from `masks_human` side-by-side PNGs, converts them to YOLO polygon format, and creates 80/20 train/val splits.

### Step 3 — Train Model *(optional — app works with pre-trained best.pt)*
```bash
python train_segmentation.py
```
Training takes 10–30 minutes on CPU, or ~5 minutes with a GPU. The best weights are saved as `best.pt`.

### Step 4 — Launch Dashboard
```bash
streamlit run app.py
```
Opens at: http://localhost:8501

## 🎨 Interactive UI Features & Customizations

### 1. Light & Dark Theme Modes
- Toggle seamlessly between a clean **Light Theme** (default) and a premium **Dark Theme** using the dropdown selector in the sidebar settings.
- Adjusts all UI elements, cards, text, and active context cards dynamically.

### 2. Multi-Image Loop Analyzer
- Upload up to **5 vehicle images** simultaneously.
- Detections run sequentially, rendering damage masks and custom report tickets with dedicated download keys for each image.

### 3. Sidebar Adjustment Controls
- **Detection Sensitivity**: Adjusts the confidence threshold for YOLO segmentation and OpenCV fallbacks (default set to `75%` for high out-of-the-box accuracy).
- **Mask Overlay Alpha**: Calibrate the opacity of the highlighted damage masks overlaid on the vehicle.

## 💡 How the Mask Extraction Works

Each image in `masks_human/` is a side-by-side PNG where:
- **Left half** = original vehicle image
- **Right half** = human-annotated damage overlay (offset by 1 pixel)

We extract the damage mask by computing the absolute pixel difference between both halves (threshold > 25). The resulting binary mask is then converted to YOLO polygon contours using OpenCV's `findContours`.
