# 🚗 DriveSure AI — Vehicle Damage Segmentation Project

A custom YOLOv8 instance segmentation system trained on human-annotated car damage masks.

---

## 📁 Project Structure

```
masks_human_project/
├── app.py                  ← Streamlit dashboard (run this!)
├── prepare_dataset.py      ← Dataset extraction & YOLO label converter
├── train_segmentation.py   ← YOLOv8-seg model training pipeline
├── predict_segmentation.py ← Inference engine (used by app.py)
├── dataset.yaml            ← Generated by prepare_dataset.py
├── best.pt                 ← Trained model weights (generated after training)
├── requirements.txt        ← Python dependencies
└── data/
    ├── images/
    │   ├── train/          ← 80% training images
    │   └── val/            ← 20% validation images
    └── labels/
        ├── train/          ← YOLO polygon .txt annotations
        └── val/
```

---

## 🚀 Quick Start

### Step 1 — Install Dependencies
```bash
pip install -r requirements.txt
```

### Step 2 — Prepare Dataset
```bash
python prepare_dataset.py
```
This extracts binary masks from `masks_human` side-by-side PNGs,
converts them to YOLO polygon format, and creates 80/20 train/val splits.

### Step 3 — Train Model *(optional — app works without this)*
```bash
python train_segmentation.py
```
Training takes 10–30 minutes on CPU, or ~5 minutes with a GPU.
The best weights are saved as `best.pt`.

### Step 4 — Launch Dashboard
```bash
streamlit run app.py
```
Opens at: http://localhost:8501

## 💡 How the Mask Extraction Works

Each image in `masks_human/` is a side-by-side PNG where:
- **Left half** = original vehicle image
- **Right half** = human-annotated damage overlay (offset by 1 pixel)

We extract the damage mask by computing the absolute pixel difference
between both halves (threshold > 25). The resulting binary mask is then
converted to YOLO polygon contours using OpenCV's `findContours`.
